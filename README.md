# Hi there, I'm Hussein Hadi 👋

## About Me

I am an English Language graduate from Baghdad, Iraq, with a growing research interest in Computational Linguistics, Natural Language Processing (NLP), Artificial Intelligence, and Large Language Models (LLMs).

My work focuses on bridging the technological gap between Iraqi Arabic and modern language technologies by developing open-source linguistic resources, datasets, and NLP tools for low-resource Arabic dialects.

I believe that language technology should be accessible to every community, regardless of the amount of existing digital resources. My long-term goal is to contribute to research that enables Iraqi Arabic to become better represented in future AI systems.

## Current Project

### Iraqi Arabic NLP Toolkit (IANLP)

An open-source project dedicated to supporting research on Iraqi Arabic through computational resources.

The project aims to build datasets, linguistic resources, preprocessing tools, normalization methods, and reusable NLP components that help researchers and developers create language technologies for Iraqi Arabic.

This initiative is part of my long-term vision to contribute to the advancement of Arabic NLP and reduce the resource gap for underrepresented dialects.
